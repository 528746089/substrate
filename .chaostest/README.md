# Usage
<!-- usage -->
```sh-session
$ npm install -g chaostest
$ chaostest COMMAND
running command...
$ chaostest (-v|--version|version)
chaostest/0.0.0 darwin-x64 node-v8.16.0
$ chaostest --help [COMMAND]
USAGE
  $ chaostest COMMAND
...
```
